# Preview Project Workflow

This workflow describes how to start the development server and preview the Altodelbosque project.

## Steps

1. **Start Development Server**:
   Run `npm run dev` to start the Next.js development server with Turbopack.
   *The server will be available at http://localhost:9002.*

2. **Verify Output**:
   Open http://localhost:9002 in your browser to verify the application is running.
